**Marshmallow Cafe Website – Frontend Project**

This project is a static frontend website developed as part of my Semester 1 coursework. The website represents a fictional brand, Marshmallow Cafe, and is built entirely using HTML5 and CSS3.

**Project Objective**

The primary objective of this project was to apply fundamental web development concepts and design principles to create a structured, visually appealing, and user-friendly website interface.

**Features**
1.Structured multi-section layout (Home, About, Menu, Gallery, Contact)
2.Semantic HTML implementation
3.Custom CSS styling for layout and design
4.Responsive design techniques using CSS
5.Navigation bar and footer integration
5.Clean and organized code structure

**Technologies Used**
1.HTML5
2.CSS3

**Learning Outcomes**
1.Understanding of semantic HTML elements
2.Implementation of CSS styling and layout techniques
3.Basic responsive web design concepts
4.Improved knowledge of webpage structuring and UI design

This project demonstrates my foundational understanding of frontend web development and the ability to design and structure a static website from scratch.
